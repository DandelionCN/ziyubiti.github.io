更换电脑时，如何更新Hexo Blog？

2016-07-11

method 1: github
在原电脑将Hexo服务端文件全部提交到github, 在新电脑拉出github仓库，进行正常Hexo操作即可。
method 2: copy file
首先按照搭建hexo的过程一步步重新在新电脑上操作，之后只要用原电脑的scaffolds, source, themes 和 _config.yml替换新生成的文件就行了；为方便可以将服务端整个文件夹拷贝下来。

百望山游记

2008-09-01

　　周日去爬百望山了，说爬可能并不太恰当，因为实在是一个很低的山包，海拔210米。
　　不过爬上山顶后，俯瞰北京城，会惊讶于北京竟然这么小，北边昌平的山峰清晰可见，东南国贸的高楼层峦叠嶂，颐和园和圆明园是围在一圈建筑中的绿色盆地，盘古大观与北边高楼之间一定是那未曾领略过的奥林匹克公园；揽枫亭上，西边太阳的光辉令人感到温暖惬意，落日的余晖让漫山绿色涂上了一层金色的光辉；当金秋来临，层林尽染，漫山红叶映夕阳一定非常壮观。
　　清新的空气使人心旷神怡，开阔的视野让人心中又生起万丈豪情，所有的不快、郁闷都不见了，不禁觉得生活是多么的美好啊！

2004年遗老们给溥仪拟的谥号和庙号

2008-03-12

　　恭宗愍皇帝，讳爱新觉罗·溥仪（公元1906－1967年,宣宗成皇帝曾孙，乃宣宗第七子醇亲王奕譞之子载沣之长子。
　　2004年，皇室後族依宣国公建议，上庙号为恭宗，谥号愍皇帝，尊骨灰奉安处为献陵。并尊婉容为孝恪愍皇后（淑妃文秀已被愍帝废为庶人），谭玉龄为明贤皇贵妃（骨灰葬处为祥妃园寝），李玉琴为敦肃福贵妃（2001年薨，73岁），李淑贤为孝睿愍皇后。
